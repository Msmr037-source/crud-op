# 🚀 CRUD App (Node.js + MongoDB + Glass UI)

A simple and modern **CRUD (Create, Read, Update, Delete)** web
application built using:

-   ⚙️ Node.js (Express)
-   🍃 MongoDB (Mongoose)
-   🎨 HTML, CSS (Glassmorphism UI)
-   🌐 Vanilla JavaScript (Fetch API)

------------------------------------------------------------------------

## ✨ Features

-   ➕ Add new users (Name & Email)
-   📄 View all users
-   ❌ Delete users
-   🔄 Update users (API ready)
-   💎 Modern Glassmorphism UI
-   ⚡ Real-time UI updates

## 🛠️ Installation & Setup

### 1️⃣ Clone the repository

    git clone https://github.com/your-username/crud-app.git
    cd crud-app

### 2️⃣ Install dependencies

    npm install

### 3️⃣ Start MongoDB

    mongod

### 4️⃣ Run the server

    node server.js

### 5️⃣ Open in browser

    http://localhost:3000

------------------------------------------------------------------------

## 🔗 API Endpoints

  Method   Endpoint      Description
  -------- ------------- ---------------
  POST     /add          Add new user
  GET      /users        Get all users
  DELETE   /delete/:id   Delete user
  PUT      /update/:id   Update user
